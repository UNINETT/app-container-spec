# Service: DNS Hostname

Identifier: `host`


Request specification:

```json
{
	"host": {
		"host": "myapplication.uninett-apps.no"		
	}
}
```


Response specification:

```json
{
	"host": {
		"host": "myapplication.uninett-apps.no"		
	}
}
```

Exposed through Environment variable:

* `HOST=myapplication.uninett-apps.no`
